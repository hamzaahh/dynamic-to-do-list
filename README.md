# TaskFlow — Dynamic To-Do List Application

**Experiment 4 | Web Development Assignment**

---

## Project Overview

TaskFlow is a fully client-side dynamic to-do list built with plain HTML5, CSS3, and vanilla JavaScript. It covers all six tasks outlined in the assignment rubric, including the bonus task.

---

## Features

| Task | Feature | Status |
|------|---------|--------|
| 1 | Project setup: folder, HTML, CSS, JS with header comments | ✅ |
| 2 | Input field, Add Task button, task list (`<ul>`) | ✅ |
| 3 | CSS styling: input/buttons, task layout, visual separation | ✅ |
| 4 | Dynamic task addition, empty-input validation | ✅ |
| 5 | Inline Edit and Delete buttons for each task | ✅ |
| 6 | Bonus — Checkbox completion, strikethrough, live counters, filter tabs, Clear Completed | ✅ |

---

## How to Run

1. Extract (unzip) the folder
2. Open `dynamic_todo_list` in VS Code
3. Right-click `index.html` → **Open with Live Server**, or just double-click `index.html`

No build tools or npm required.
